**Python Foundations — 110 Problems**
A structured practice repository covering Python fundamentals, solved as the first step of my journey toward Data Engineering.

Why This Exists
Before working with pipelines, distributed systems, or large-scale data, the fundamentals have to be solid. This repository documents that groundwork — 110 problems covering core Python from scratch, organized by topic and solved without shortcuts.



Topics Covered

Variables, operators, and control flow
Loops — for, while, nested
Functions and recursion
Strings — manipulation, searching, frequency analysis
Lists, dictionaries, and sets
Number theory — primes, GCD, LCM, Armstrong, palindromes
Pattern printing
Searching — linear and binary
Stack logic and balanced expressions
